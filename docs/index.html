The loop

  1. Ingest — nightly box scores scraped politely (75% of the documented rate cap, exponential backoff, resumable checkpoints).
  2. Rebuild — every feature is a shift(1) window: game t sees only games ≤ t−1. Walk-forward CV only.
  3. Monitor — PSI on phase-invariant differential features + a Brier z-test against the model's own holdout ledger.
  4. Retrain (gated) — drift signal or 150+ new games; a challenger must beat the champion on shared holdout to be promoted.
  5. Predict — tonight's slate: win probability, fair moneyline, projected spread, player PI.
